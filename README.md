# Eco-Track — [Ridho Satrio Nugroho]

## Tentang Project
Eco-Track adalah aplikasi web kalkulator jejak karbon harian yang membantu pengguna menghitung emisi CO₂ dari aktivitas transportasi dan penggunaan energi sehari-hari.

## Masalah Iklim yang Ingin Disoroti
Emisi karbon dari aktivitas sehari-hari, seperti berkendara dan penggunaan alat elektronik. Melalui Eco-Track, pengguna dapat menghitung seberapa besar emisi karbon yang dihasilkan dari aktivitas yang telah mereka lakukan selama 1 hari. Dengan adanya aplikasi web Eco-Track ini, saya harap pengguna dapat termotivasi untuk berusaha mengurangi jejak karbon mereka.
